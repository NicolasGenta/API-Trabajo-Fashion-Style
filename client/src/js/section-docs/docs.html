Documentación de la API

Introducción

Esta API proporciona acceso a una colección de recursos y permite realizar operaciones básicas de lectura y escritura. Los métodos admitidos son GET para recuperar recursos, GET by ID para obtener un recurso específico y POST para crear nuevos recursos.

Base URL

"http://localhost:3000/productos"
