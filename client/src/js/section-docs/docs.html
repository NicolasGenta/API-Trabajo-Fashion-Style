Documentación de la API

Introducción

Esta API proporciona acceso a una colección de recursos y permite realizar operaciones básicas de lectura y escritura. Los métodos admitidos son GET para recuperar recursos, GET by ID para obtener un recurso específico, POST para crear nuevos recursos, PUT para modificación de recursos y DELETE para eleiminar un recurso especifico.
